# Movie-Store

#### Projeto (Front-End) de uma loja de filmes utilizando a API TMDB https://developers.themoviedb.org/3

### Tecnologias utilizadas;

- React.JS
- Typescript
- React-Router-Dom
- Redux-Thunk
- React-Hook-Form
- Yup
- Styled-Components
- Axios
- React-Icons
- React-Toastfy
